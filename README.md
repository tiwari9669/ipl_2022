**Project Overview**
Title: IPL 2022 Capstone Project Objective: The project aims to analyze match-level data from the Indian Premier League (IPL) 2022 season. The goal is to derive meaningful insights regarding match outcomes, team dynamics, toss decisions, player performance (batting and bowling), and venue statistics.


**Dataset Features (Columns)**
The dataset consists of 74 rows and 20 columns, containing the following variables:

Match Details: match_id, date, venue, stage (Group/Playoff/Final).

Teams: team1, team2.

Toss: toss_winner, toss_decision (Bat/Field).

Scores & Wickets: first_ings_score, first_ings_wkts, second_ings_score, second_ings_wkts.

Outcomes: match_winner, won_by (Runs/Wickets), margin.

Player Stats: player_of_the_match, top_scorer, highscore, best_bowling, best_bowling_figure.
